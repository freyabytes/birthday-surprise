# Birthday Surprise Website 💗

A small interactive birthday website created as a personal surprise for someone special.

## Features

* Interactive envelope opening
* Birthday email screen
* Personal birthday letter
* Floating heart animations
* Confetti effect
* Mobile-friendly responsive design

## Built With

* HTML
* CSS
* JavaScript
